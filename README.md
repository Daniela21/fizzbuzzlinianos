# fizzBuzz for Linianos

This program prints all the numbers from 1 to 100 ...is a fizzBuzz for linio.

## To start

### Installation

fizzBuzz for Linianos requires [getcomposer.org](https://getcomposer.org/) to run.

```sh
composer install
```


## To use

```sh
composer app
```

## To run all tests

```sh
composer tests
```
